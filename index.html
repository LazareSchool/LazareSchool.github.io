Installation

To install the necessary dependencies, follow these steps:

  1. Clone the repository: 
    git clone https://github.com/LazareSchool/csa-mp3-pd3.git
cd csa-mp3-pd3
  2. Install dependencies: 
    pip install -r requirements.txt
